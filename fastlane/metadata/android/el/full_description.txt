Το ForgetMeNot είναι μια εφαρμογή για την απομνημόνευση πληροφορίας με τη χρήση εκπαιδευτικών καρτών. Η απλότητα, η χρηστικότητα και η ταχύτητα λήφθηκαν υπόψη κατά τη διαδικασία ανάπτυξης αυτής της εκπαιδευτικής εφαρμογής. Αυτή η εφαρμογή παρέχει εξαιρετικές συνθήκες για την επίτευξη υψηλής αποδοτικότητας στην απομνημόνευση.


Τα χαρακτηριστικά που υποστηρίζονται είναι:

    • Εισαγωγή/εξαγωγή αρχείων.
    • Υποστήριξη CSV, κείμενο χωρισμένο με Tab ή οποιουδήποτε άλλου είδους Διαχωριζόμενης με Οριοθέτη τιμής.
    • Διαστήματα (Βηματική επανάληψη). Μπορείτε να καθορίσετε το δικό σας καθεστώς χρονικού διαστήματος για κάθε τράπουλα. Επίσης είναι δυνατό να υποδείξετε τον δικό σας αλγόριθμο για το πως οι απαντήσεις επηρεάζουν την επιλογή του επόμενου διαστήματος.
    Πολλές μέθοδοι εξέτασης. Υπάρχοι οι 'Αυτοέλεγχος', 'Έλεγχος με παραλλαγές', 'Ορθογραφικός έλεγχος'.
    • Εκφώνηση του κειμένου μέσω TTS. Μπορείτε να επιλέξετε γλώσσες για τις ερωτήσεις και τις απαντήσεις και να ενεργοποιήστε την αυτόματη εκφώνηση τους.
    • Κρύψιμο του κειμένου μίας ερώτησης για την τόνωση της βελτίωσης τις ικανότητας ακουστικής κατανόησης, που είναι πολύ χρήσιμη στην εκμάθηση μίας ξένης γλώσσας.
    • Αντιστροφή κάρτας.
    • Υποδείξεις με τη μορφή συγκάλυψης γραμμάτων.
    • 'Χρονομέτρου κινήτρου' το οποίο θα σας κάνει να συγκεντρωθείτε στη μελέτη σας (προαιρετικά).
    • Αποθήκευση των ρυθμίσεων μίας τράπουλας ως προεπιλογές και επαναχρησιμοποίησή τους, για να αποφύγετε επαναλαμβανόμενες ενέργειες στις ρυθμίσεις.
    • Επεξεργασία και αναζήτηση καρτών κατά τη διάρκειας της εξέτασης.
    • 'Λειτουργία περπατήματος' το οποίο σας επιτρέπει να εξασκηθείτε χωρίς να κοιτάτε την οθόνη.
    • 'Λειτουργία αυτόματης αναπαραγωγής'. Σε αυτή τη λειτουργία οι ερωτήσεις και οι απαντήσεις προφέρονται διαδοχικά. Μπορείτε να συνδυάσετε τις δικές σας δραστηριότητες και την επανάληψη του διδακτικού υλικού.
    • Κατάλογος προκατασκευασμένων τραπουλών. Ο κατάλογος περιέχει πολλές τράπουλες για την εκμάθηση γλωσσών, ο οποίο περιλαμβάνει σύνολα λέξεων, θεματικές λέξεις και φράσεις και ολόκληρες προτάσεις.
    • Ομαδοποίηση των τραπουλών σε ξεχωριστές λίστες.
    • Παραμετροποίηση της εμφάνισης των καρτών.
    • Σκοτεινό θέμα.
    • Backup.


Τι κάνει το ForgetMeNot τόσο αποτελεσματικό για την απομνημόνευση:

    - Απουσία διαφημίσεων. Μπορείτε να χειριστείτε πολλές κάρτες σε σύντομο χρόνο, λόγω της απλότητας και της ευκολίας του περιβάλλοντος χρήστη. Τίποτα δε σας εμποδίζει από τη μελέτη.
    - Η εξάσκηση είναι σχεδιασμένη, έτσι ώστε τελικά να μάθετε την απάντηση. Αν η απάντηση σας είναι λάθος, η κάρτα αναβάλλεται για το τέλος της λίστας μέχρι να την απαντήσετε σωστά.
    - Διαστήματα (Βηματική επανάληψη). Η διακεκομμένη επανάληψη μειώνει την πιθανότητα να ξεχάσετε υλικό που δεν συναντάται σε σημαντικό χρονικό διάστημα. Περιλαμβάνει, επίσης, την ενεργό ανάκληση ήδη διδαγμένου υλικού, που βελτιώνει τη μάθηση.
    - Δυνατότητα να επεξεργαστείτε μία κάρτα κατά τη διάρκεια της εξάσκησης. Αυτό σας επιτρέπει όχι μόνο να διορθώσετε τα λάθη σας, αλλά και να ανακατασκευάσετε κάρτες που είναι δύσκολες στην απομνημόνευση. Για παράδειγμα, μπορείτε να προσθέσετε μια πρόσθετη συσχέτιση σε παρένθεση.
    - Φωνητική συνοδεία. Αυτό είναι ιδιαίτερα χρήσιμο κατά την εκμάθηση μίας γλώσσας γιατί βελτιώνει τις ικανότητες ακρόασης, βοηθά στη συγκέντρωση στη μελέτη και ευνοεί την απομνημόνευση. Μπορείτε ακόμα να ρυθμίσετε μία τράπουλα να εξετάζει μία ερώτηση κατ' εξαίρεση ακουστικά.
    - Λειτουργίες που σας επιτρέπουν να μην εξασκείστε μόνο στον ελεύθερο σας χρόνο. Η 'Λειτουργία Περπατήματος' σας επιτρέπει να εξασκηθείτε κατά το περπάτημα. Η 'Λειτουργία Αυτόματης Αναπαραγωγής' σας επιτρέπει να ακούτε τις ερωτήσεις και τις απαντήσεις όταν τα χέρια και τα μάτια σας είναι απασχολημένα, αλλά τα αυτιά σας είναι διαθέσιμα. :)
    - Βοηθήματα συγκέντρωσης. Η 'Χρονομέτρηση κινήτρου' σας υπενθυμίζει όταν έχει χαθεί η προσοχή σας. Λειτουργία πλήρους οθόνης κρύβει τη γραμμή κατάστασης, η οποία μπορεί να είναι μια πρόσθετη πηγή αποσπάσματος της προσοχής.
